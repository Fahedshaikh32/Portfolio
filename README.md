## 👤 About Me

I'm an Electronics & Telecommunication Engineer working across **embedded systems, IoT, and low-level C programming**. My projects range from real-time microcontroller firmware and CAN-bus automotive systems to Linux systems programming and networked C applications — I like understanding how things work down to the register and the byte.

This repository is my portfolio: a curated index of my strongest work, along with my resume for anyone hiring or collaborating.
